# contexto

Em dispositivos móveis, a topbar não é exibida. Por isso, não é possível navegar para as páginas Posts e Sobre. Assim, faz-se necessária a correção da exibição desses links da topbar no mobile.

# requisitos

- Exibir os links inicio, artigos, sobre em dispositivos com telas menores
- manter a tradução conforme já funciona
- não alterar o funciomamento em dispositivos com telas maiores
- APENAS a exibição dos links deve ser corrigida, NENHUM conteúdo deve ser alterado
- o suporte a i18n deve ser mantido
