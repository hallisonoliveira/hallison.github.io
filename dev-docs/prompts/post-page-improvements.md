# Contexto

A página dos posts está com o design desagradável e está fugindo do layout geral do site. É necessário rever a UI/UX do layout das páginas de post a fim de melhorar seu visual e usabilidade.

# Melhorias
- Reduzir o tamanho do título
- Reduzir o tamanho da imagem
- O texto não possui espaçamento entre paragrafos
- o texto não possui tabulação para inicio de paragrafo
- as imagens adicionadas no corpo do post não possuem margens
- a legenda das imagens deve ser centralizada abaixo da imagem
- não há espaçamento antes nem depois das imagens
- a lista "leitura relacionada" exibe o próprio post em exibição
- a exibição de links no corpo do post não possui fonte e cor específica. A fonte e cor é a mesma do resto do texto.
